1. Introduction

The Microsoft Installer packages are available in a number of localized translations. To specify a specific localization or language use the TRANSFORMS property when invoking msiexec to install.

See the Tor Project Translation site for additional information.

1.1. Supported Package Languages


en, cs, de, es, fa, fi, fr, he, it, nl, pl, pt, ro, ru, sv
   

Ex. torvm-de.msi


   

1.2. Supported Translation Transforms
