1. Introduction

1.1. Masking Timezone

It is often desirable to mask the local timezone configured on the Windows host. The signifigance of risk for timezone leakage is difficult to quantify; using the GMT timezone without any daylight savings adjustment provides the least amount of information and is recommended.

2. Adjust Timezone

2.1. Adjust Timezone As Admin

Right click on the clock dock icon and select the "Adjust Date/Time" option.



   

Time zone settings are on the second tab.



   

Select the "GMT Greenwich Mean Time" timezone.



   

Disable daylight savings time adjustment if necessary.



   

Select "Apply" option to make timezone changes take effect.



   

2.2. Confirm Limited User Timezone

Now any software running on the limited user account will only be able to read localtime in GMT.
