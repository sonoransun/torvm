1. Introduction

1.1. Defense In Depth With Reduced Privilege Accounts

Many applications for Windows operating systems fail to implement even the most basic separation of priveleges between applications. The ability to restrict the rights of untrusted applications by user is present on all supported versions of Windows. Preventing rogue code from escalating an application flaw into full host compromise is a useful defensive technique despite the additional effort required.

2. Create Limited User

2.1. User Accounts Utility

Navigate to the control panel.



   

Select "Create a new account" option.



   

Provide a nickname for the anonymous limited user account.



   

The "Limited" account type is the reduced privilege user option.



   

Create the account.



   

The new user is added to the list of accounts on the local system.
